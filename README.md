# Veri Görselleştirme
 Pyhton'da veri görselleştirme; 
 ### Grafikler
   - Histogram
      * Sayısal değişkenler vardır.       
   - Kutu (*box plot*)
       * Sayısal değişkenler vardır.
   - Sütun (*bar plot*)
       * Kategorik değişken vardır. Ve bu değişkenerin sınıf dağılımlarına erişebiliriz.
   - Violin
       * Sayısal değişkenler vardır. 
 - Korelasyon
      * Sayısal değişkenler vardır. Ve değişkenler arasında ilişkiler
 - Isı (*heatmap*)
